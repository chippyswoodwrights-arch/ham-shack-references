## Ham Radio Outlet (HRO)

- Knowledgeable staff — they are hams.
- Walk-in retail, no account needed.
- **KO6NOI area:** 4813 Auburn Blvd, Sacramento CA 95841 — (916) 659-7373
  Hours: Mon/Wed–Sat 10:00 AM – 5:30 PM (CLOSED Tue & Sun)

## Bridgecom

- **WARNING: DO NOT BUY FROM HERE.**
- Locks radios after programming — must return to Bridgecom for any programming changes.
- Anti-operator practice. Flagged 2026-06-17.
- Buy the same radio elsewhere (HRO, Amazon, GigaParts).

---

## RT Systems

- Makes radio-specific programming software + cables. Software and cable are both per-radio —
  no universal package. Expect ~$45–50 per radio model.
- USB-57B cable connects to FT-70DR SP/MIC jack (3.5mm TRRS, top of radio) — bypasses
  DATA jack and button dance entirely. Much easier than ADMS-10.
- RT Systems does NOT make software for Anytone. For Anytone: use CPS or CHIRP (partial).

---

## Yaesu ADMS-10 (avoid for FT-70DR)

Known failures documented 2026-06-17:
- Memory bank management broken — cannot properly organize banks
- C4FM/digital memory parameters incomplete
- CSV import broken (undocumented requirements, silent failures)
- Frequency display errors after read
- Set mode settings lost between version upgrades

**Use RT Systems ADMS-70D instead.** ~$45-50 total with USB-57B cable.

## Anytone CPS (for Fred's DMR radio)

- Chinese software. Security concern — run in VirtualBox VM with no network access if self-programming.
- **Better option: let HRO program it** under their lifetime programming policy. Avoids CPS entirely.
- Fred must register at RadioID.net before any DMR programming is possible.

---

## RadioID.net

- **Required for DMR.** Register callsign to get a DMR ID number before programming any DMR radio.
- Free registration. Takes 24-48 hours to process.
- radioID.net

- Legitimate community resource maintained by professional engineers.

## Talley LLC

- Western US wireless infrastructure distributor. Primary market is cell tower crews and
  commercial wireless contractors.
- Sells to general public at list price — no account, no minimums, will-call at warehouse.
- Best source for LMR-400/LMR-240 cable. PL-259 in stock but secondary focus — call ahead.
- Web catalog incomplete — subcategory and search pages often "Not Yet Available." Call the counter.
- See vendors.md for pricing details and locations.
